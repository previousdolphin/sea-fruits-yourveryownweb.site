# Sea Fruits

Sea Fruits is a fine-dining restaurant website showcasing impossible hybrid creatures—ocean and orchard merged into one. The site features interactive Three.js 3D renderers of signature creatures (Lobstrawberry, Tunana, Salmberry), a full seasonal menu with five newly invented hybrid dishes, reservation system, and visit information. Built with vanilla JavaScript and semantic HTML.
